Все просто

Пользователи соревновнуются между собой в серии экспрессов. Чем больше экспрессов сделано - тем больше игрок получит поинтов которые поднимут его в рейтинге.

Каждый месяц список лидеров будет обнуляться. Сортировка пользователь будет происходить по общей сумме заработанных кэф.

Условия акции

Моментальное пополнение счета любым из предложенных способов!
